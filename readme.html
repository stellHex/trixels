Canvas controls
---------------------
LCLICK+DRAGDraw with foreground color
RCLICK+DRAGDraw with background color
MCLICK+DRAGDraw with tertiary color
SHIFT+CLICK+DRAG Select if first trixel not already selected, deselect otherwise
ALT+LCLICKSet foreground color
ALT+RCLICKSet background color
ALT+MCLICKSet tertiary color
CCopy selection
XCut selection, replacing with background color
VReduplicate the previously copied section

Copying controls
---------------------
UP/DOWN/LEFT/RIGHT Move the copied section
COMMA/PERIODRotate the copied section left/right
C/VPaste

Color Controls
---------------------
LCLICK (on actives) Set that color with HVS
SPACECycle foreground, background, and tertiary colors
LCLICK (on palette)Switch foreground color to that palette color
ALT+LCLICKSet palette color
1-0Switch the foreground color to corresponding palette color
